  [Summeray]
    This tool can be used to render a scene to an cubemap asset.
--------------------------------------------------------------------------
  [Environment]
    This tool applies to Unity 5.0, .Net Framework 3.0 or above version.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Cubemap Renderer" in Unity editor menu bar
    and click it or press combination key Alt+C to open the editor window.
    Create a camera as the "RenderCamera".
    Click the "Render" button to open the dialog of save new render cubemap.
    Select a path and input a file name to dialog and save.
